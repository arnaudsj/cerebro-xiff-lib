Thank you for downloading the XIFF 3 Beta!

XIFF is an open source ActionScript 3/Flex API for instant messaging and presence using the XMPP protocol. We would love to have your feedback on how it can be improved. In particular, the API has changed in many ways in this beta release, and feedback is encouraged on those changes.
